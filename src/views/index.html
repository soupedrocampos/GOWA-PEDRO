Agentes de IA

Configure respostas automáticas por dispositivo: via n8n (webhook) ou LLM direto (Ollama, OpenAI, Groq…)

Como funciona

1 — n8n (Webhook)
Configure o webhook na aba Device setup. O ROBIT encaminha cada mensagem recebida ao n8n, que processa com IA e chama /send/message de volta.

2 — LLM Direto
O ROBIT chama a API do LLM a cada mensagem recebida e responde automaticamente — sem precisar do n8n.

Providers suportados

Ollama — modelos locais (llama3, mistral, qwen…)
OpenAI — GPT-4o, GPT-4o-mini…
Groq — llama-3.1 ultrarrápido
Custom — qualquer API OpenAI-compatível
Dicas
  • O LLM só responde a chats individuais — grupos são ignorados.
  • Use o System Prompt para definir a personalidade do agente.
  • Clique em Testar LLM antes de salvar para verificar a conexão.
  • n8n e LLM podem coexistir no mesmo dispositivo.
